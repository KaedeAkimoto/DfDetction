# YOLO11 缺陷检测改进方案 (6v-13v)

## 概述

基于 YOLO11n 基线模型，针对缺陷检测任务设计了 4 种改进方案，分别从注意力机制、卷积算子、特征融合、损失函数等维度进行优化。

| 方案 | 核心改进 | 参数量 | 损失函数 |
|------|---------|--------|---------|
| 6v | EMA 注意力 + GSConv 轻量 Neck | ~3.3M | WIoU v3 |
| 7v | AKConv 可变形卷积 + BiFormer 注意力 | ~3.1M | Shape-IoU |
| 8v | C2fDBB 多分支卷积 + NWD 小目标损失 | ~3.3M | NWD |
| 9v | C2PSA_MSDA 多尺度空洞注意力 + DyHead | ~3.5M | MPDIoU |
| 10v | C2fLSK 大核注意力 + PConv 轻量颈部 | 3.24M | Focaler-MPDIoU |
| 11v | C2fDCNv3 可变形卷积 + AKConv 颈部 | 3.10M | AIoU |
| 12v | C2fA2 区域注意力 + DyHeadv3 动态头 | 4.20M | Focaler-NWD |
| 13v | C2fRELAN 残差网络 + CSDNHead 自适应头 | 5.45M | AIoU |

---

## 6v 方案: EMA + GSConv + WIoU

### 设计思路

轻量级优化方案，在保持低计算量的前提下提升特征表达能力。

### 改进模块

| 模块 | 类型 | 说明 |
|------|------|------|
| C2fEMA | 注意力 | 在 C2f 中嵌入 Efficient Multi-Scale Attention，跨维度特征加权 |
| C2fGSConv | 卷积 | 在 Neck 中使用 Grouped Shuffle Convolution，降低参数量同时保持精度 |
| WIoU v3 | 损失 | 动态非单调聚焦机制，聚焦中等质量锚框，减少低质量样本干扰 |

### 网络结构

```
Backbone: Conv → Conv → C3k2 → Conv → C3k2 → Conv → C2fEMA×2 → Conv → C2fEMA×2 → AIFI → Conv → C2PSA×2
Neck:     FPN+PAN 结构，全部替换为 C2fGSConv
Head:     4尺度检测头 (P2-P5)
```

### 训练配置

```python
model='yolo11_fix_6.yaml'
batch=36
cos_lr=True
label_smoothing=0.1
weight_decay=0.001
dropout=0.1
iou_type='WIoU'
```

### 适用场景

- 需要轻量级改进，计算资源有限
- 各类缺陷的通用检测

## 7v 方案: AKConv + BiFormer + Shape-IoU

### 设计思路

内存优化方案，使用可变形卷积和双向路由注意力，精简检测头为 2 尺度以降低显存占用。

### 改进模块

| 模块 | 类型 | 说明 |
|------|------|------|
| AKConv | 卷积 | Alterable Kernel Convolution，可变形卷积核，自适应目标形状 |
| C2fBiFormer | 注意力 | 嵌入 Bi-Level Routing Attention，在非均匀区域进行稀疏注意力计算 |
| Shape-IoU | 损失 | 关注边界框形状和尺度，对不规则缺陷更敏感 |

### 网络结构

```
Backbone: AKConv×6 → C3k2×2 → C2fBiFormer×2 → AIFI → AKConv → C2PSA
Neck:     FPN+PAN 结构，C2fBiFormer 特征融合
Head:     2尺度检测头 (P4-P5)，减少一个检测尺度节省显存
```

### 训练配置

```python
model='yolo11_fix_7.yaml'
batch=16
amp=True              # 混合精度训练，显存减半
cos_lr=True
label_smoothing=0.1
weight_decay=0.001
dropout=0.1
iou_type='ShapeIoU'
```

### 适用场景

- 显存受限环境 (如 16GB 以下显卡)
- 中大型缺陷检测 (P4-P5 尺度)

## 8v 方案: C2fDBB + NWD

### 设计思路

小目标优化方案，使用多分支卷积增强特征提取，采用 NWD 损失提升小目标检测精度。

### 改进模块

| 模块 | 类型 | 说明 |
|------|------|------|
| C2fDBB | 卷积 | Diverse Branch Block，训练时多分支结构增强感受野，推理时可合并为单路 |
| NWD | 损失 | Normalized Wasserstein Distance，对小目标位置偏差更鲁棒 |

### 网络结构

```
Backbone: Conv → Conv → C3k2 → Conv → C3k2 → Conv → C2fDBB×2 → Conv → C2fDBB×2 → AIFI → Conv → C2PSA×2
Neck:     FPN+PAN 结构，全部替换为 C2fDBB
Head:     4尺度检测头 (P2-P5)
```

### 训练配置

```python
model='yolo11_fix_8.yaml'
batch=36
cos_lr=True
label_smoothing=0.1
weight_decay=0.001
dropout=0.1
iou_type='NWD'
```

### 适用场景

- 小目标缺陷检测 (如划痕、裂纹)
- 对定位精度要求高的场景

## 9v 方案: C2PSA_MSDA + DyHead + MPDIoU

### 设计思路

高性能优化方案，结合多尺度空洞注意力和动态检测头，最大化特征表达能力。

### 改进模块

| 模块 | 类型 | 说明 |
|------|------|------|
| C2PSA_MSDA | 注意力 | Multi-Scale Dilated Attention，多膨胀率空洞注意力捕获多尺度上下文 |
| DyHeadBlock | 检测头 | Dynamic Head，尺度-空间-任务三重注意力动态加权特征 |
| MPDIoU | 损失 | Minimum Point Distance IoU，基于边界框顶点距离的损失函数 |

### 网络结构

```
Backbone: Conv → Conv → C3k2 → Conv → C3k2 → Conv → C2PSA_MSDA×2 → Conv → C2PSA_MSDA×2 → AIFI → Conv → C2PSA_MSDA×2
Neck:     FPN+PAN 结构，C3k2 + DyHeadBlock 组合
Head:     4尺度检测头 (P2-P5)，每尺度后接 DyHeadBlock
```

### 训练配置

```python
model='yolo11_fix_9.yaml'
batch=36
cos_lr=True
label_smoothing=0.1
weight_decay=0.001
dropout=0.1
iou_type='MPDIoU'
```

### 适用场景

- 追求最高精度，计算资源充足
- 多尺度缺陷共存场景

## 10v 方案: C2fLSK + PConv + Focaler-MPDIoU

### 设计思路

轻量级特征提取优化方案，结合大核选择性注意力（LSK）和部分卷积（PConv），在保持低计算量的同时增强多尺度特征表达能力。

### 改进模块

| 模块 | 类型 | 说明 |
|------|------|------|
| C2fLSK | 注意力 | Large Selective Kernel 大核注意力，通过大卷积核捕获更广范围的上下文信息 |
| C2fPConv | 卷积 | Partial Convolution，仅对部分输入通道进行卷积，大幅降低 FLOPs 和参数量 |
| Focaler-MPDIoU | 损失 | 结合 Focaler 动态区间聚焦机制和 MPDIoU 点距离度量，聚焦困难样本 |

### 网络结构

```
Backbone: Conv → Conv → C3k2 → Conv → C3k2 → Conv → C2fLSK×2 → Conv → C2fLSK×2 → AIFI → Conv → C2PSA×2
Neck:     FPN+PAN 结构，全部替换为 C2fPConv（P2-P5 四尺度）
Head:     4尺度检测头 (P2-P5)
```

### 训练配置

```python
model='yolo11_fix_10.yaml'
batch=48
cos_lr=True
label_smoothing=0.1
weight_decay=0.001
dropout=0.1
iou_type='FocalerMPDIoU'
focaler_d=0.0
focaler_u=0.95
```

### 适用场景

- 需要大感受野捕获全局上下文
- 各类缺陷的通用检测

## 11v 方案: C2fDCNv3 + AKConv + AIoU

### 设计思路

可变形特征提取方案，使用 DCNv3 可变形卷积增强几何形变建模能力，配合 AKConv 自适应核卷积和 3 尺度检测头。

### 改进模块

| 模块 | 类型 | 说明 |
|------|------|------|
| C2fDCNv3 | 卷积 | Deformable Convolution v3，可变形卷积 v3，增强对不规则形状目标的特征提取 |
| C2fAKConv | 卷积 | Alterable Kernel Convolution，自适应卷积核形状，匹配目标几何结构 |
| AIoU | 损失 | Adaptive IoU，自适应 IoU 损失，根据目标尺度动态调整损失权重 |

### 网络结构

```
Backbone: Conv → Conv → C3k2 → Conv → C3k2 → Conv → C2fDCNv3×2 → Conv → C2fDCNv3×2 → AIFI → Conv → C2fDCNv3×2
Neck:     FPN+PAN 结构，C2fAKConv 特征融合（P3-P5 三尺度）
Head:     3尺度检测头 (P3-P5)
```

### 训练配置

```python
model='yolo11_fix_11.yaml'
batch=32
amp=True
cos_lr=True
label_smoothing=0.1
weight_decay=0.001
dropout=0.1
iou_type='AIoU'
```

### 适用场景

- 不规则形状缺陷检测
- 需要几何形变建模的场景

## 12v 方案: C2fA2 + DyHeadv3 + Focaler-NWD

### 设计思路

区域注意力与动态检测头结合的高性能方案，使用 Area Attention (A²) 捕获区域级特征关联，配合 DyHeadv3 实现尺度-空间-任务三重动态加权。

### 改进模块

| 模块 | 类型 | 说明 |
|------|------|------|
| C2fA2 | 注意力 | Area Attention (A²)，区域级注意力机制，对特征图分区域建模关联性 |
| DyHeadv3 | 检测头 | Dynamic Head v3，改进版动态头，集成尺度感知、空间感知、任务感知三重注意力 |
| Focaler-NWD | 损失 | 结合 Focaler 动态区间聚焦和 Normalized Wasserstein Distance，对小目标更友好 |

### 网络结构

```
Backbone: Conv → Conv → C3k2 → Conv → C3k2 → Conv → C2fA2×2 → Conv → C2fA2×2 → AIFI → Conv → C2PSA×2
Neck:     FPN+PAN 结构，C3k2 + DyHeadv3 组合（P2-P5 四尺度）
Head:     4尺度检测头 (P2-P5)，每尺度后接 DyHeadv3
```

### 训练配置

```python
model='yolo11_fix_12.yaml'
batch=36
cos_lr=True
label_smoothing=0.1
weight_decay=0.001
dropout=0.1
iou_type='FocalerNWD'
focaler_d=0.0
focaler_u=0.95
```

### 适用场景

- 多尺度缺陷共存场景
- 小目标密集检测

## 13v 方案: C2fRELAN + CSDNHead + AIoU

### 设计思路

残差聚合与上下文自适应检测头的高性能方案，使用 R-ELAN 残差聚合网络增强梯度流动，配合 CSDNHead 实现全局-局部-细节三重上下文门控融合。

### 改进模块

| 模块 | 类型 | 说明 |
|------|------|------|
| C2fRELAN | 卷积 | R-ELAN (Residual Efficient Layer Aggregation Network)，残差聚合网络，增强梯度流动和特征复用 |
| CSDNHead | 检测头 | Context-Gated Self-Adaptive Detection Network head，结合 Block Attention（全局）、Neighbor Attention（局部）、Deformable Attention（细节）三重上下文，通过自适应门控融合 |
| AIoU | 损失 | Adaptive IoU，自适应 IoU 损失 |

### 网络结构

```
Backbone: Conv → Conv → C3k2 → Conv → C3k2 → Conv → C2fRELAN×2 → Conv → C2fRELAN×2 → AIFI → Conv → C2PSA×2
Neck:     FPN+PAN 结构，C3k2 + CSDNHead 组合（P2-P5 四尺度）
Head:     4尺度检测头 (P2-P5)，每尺度后接 CSDNHead
```

### 训练配置

```python
model='yolo11_fix_13.yaml'
batch=32
cos_lr=True
label_smoothing=0.1
weight_decay=0.001
dropout=0.1
iou_type='AIoU'
```

### 适用场景

- 追求最高精度，计算资源充足
- 复杂背景下的缺陷检测
- 需要多尺度上下文融合的场景
